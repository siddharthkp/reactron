import ButtonExamples from "reactron/src/button/examples";

## Examples

<ButtonExamples.Simple />

```jsx
<Button appearance="info">Click me</Button>
```

### Appearances

There are multiple appearances for `Button`, dependending on the context.

<ButtonExamples.Styles />

```jsx
<Button appearance="default">Click me</Button>
<Button appearance="primary">Click me</Button>
<Button appearance="secondary">Click me</Button>
<Button appearance="destructive">Click me</Button>
<Button appearance="link">Click me</Button>
```

### Sizes

<ButtonExamples.Sizes />

```jsx
<Button size="small">Click me</Button>
<Button size="default">Click me</Button>
<Button size="large">Click me</Button>
```

### Loading state

<ButtonExamples.LoadingState />

```jsx
<Button appearance="default">Click me</Button>
<Button appearance="primary">Click me</Button>
<Button appearance="secondary">Click me</Button>
<Button appearance="destructive">Click me</Button>
<Button appearance="link">Click me</Button>
<Button size="small"><Spinner /> Click me</Button>
<Button size="default"><Spinner /> Click me</Button>
<Button size="large"><Spinner /> Click me</Button>
```
