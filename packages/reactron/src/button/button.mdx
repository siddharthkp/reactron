## Examples

<ButtonExamples.Simple />

```jsx
<Button appearance="info">Click me</Button>
```

### Appearances

There are multiple appearances for `Button`, dependending on the context.

<ButtonExamples.Styles />

```jsx
<Button>Click me</Button>
<Button appearance="info">Click me</Button>
<Button appearance="success">Click me</Button>
<Button appearance="warning">Click me</Button>
<Button appearance="error">Click me</Button>
<Button appearance="link">Click me</Button>
```
